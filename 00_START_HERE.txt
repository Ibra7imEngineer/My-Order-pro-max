• حمّل XAMPP من: https://www.apachefriends.org/
• ثبّت البرنامج بالطريقة العادية

الخطوة 2: نقل الملفات
───────────────────────────────────
انسخ مجلد "My Order pro" إلى:
  C:\xampp\htdocs\

الخطوة 3: تشغيل الخوادم
───────────────────────────────────
• افتح XAMPP Control Panel
• اضغط Start بجانب Apache و MySQL

الخطوة 4: إنشاء قاعدة البيانات
───────────────────────────────────
• افتح: http://localhost/phpmyadmin

الخطوة 5: تشغيل الموقع
───────────────────────────────────
• افتح: http://localhost/My%20Order%20pro/
• ابدأ الاستخدام! 🎉
